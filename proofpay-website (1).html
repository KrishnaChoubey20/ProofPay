Transaction flow

How ProofPay works

Four steps from wallet connection to a verified on-chain payroll receipt.

01

Connect Freighter

Click Connect and grant ProofPay access to your Freighter wallet on Stellar Testnet.

02

View XLM balance

Your live XLM balance is fetched from Stellar Testnet Horizon and shown in real time.

03

Send test payroll

Enter a recipient address and amount, then sign the transaction in Freighter.

04

Get your receipt

A success receipt with transaction hash and StellarExpert link appears instantly.

Technology

Built on proven tools

A lean, well-typed stack for Stellar-native development.

⚛️

React + TypeScript

Typed, component-driven UI with strict compiler settings and full ESLint coverage.

⚡

Vite

Fast HMR dev server and production build with tree-shaking for the Stellar SDK.

🔑

Freighter API v4

Browser extension wallet for Stellar — handles key management and transaction signing.

🌐

Stellar SDK v13

Official JS SDK for building transactions, reading balances, and submitting to Horizon.

🛰️

Testnet Horizon

Public Testnet API — free, no key needed, always live at horizon-testnet.stellar.org.

🔗

StellarExpert

Explorer link shown in every success receipt for independent transaction verification.
